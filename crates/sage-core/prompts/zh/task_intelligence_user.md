- **关键：若已有未完成任务涵盖相同主题，不要建议 NEW 任务**

- 不要将 [id=XX] 标记复制到任务内容中
- 存疑时输出 NONE——宁可不建议，也不要重复
- 每次最多 3 个信号
- 证据和处理方式保持简洁（各不超过 60 字）
